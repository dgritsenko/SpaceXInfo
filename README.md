___



## Permissions
- `android.permission.INTERNET`
- `android.permission.WRITE_EXTERNAL_STORAGE`
- `android.permission.READ_EXTERNAL_STORAGE`
- `android.permission.SET_WALLPAPER`

## SpaceX REST API
* Base Url  `https://api.spacexdata.com/v4`
* Rockets `GET /rockets`
* Crew  `GET /crew`
* Ships `GET /ships`
* Dragonds `GET /dragons`
